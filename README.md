* 首次安装教程

 > 1. 安装mysql 8 ，账号root，密码 自定义（修改 snowy_dev.properties）
  2. 使用 init-vx.x.x.sql 初始化数据库（使用最新的SQL，后续若数据变化，重新生成init.sql，覆盖原有数据）
  3. 运行App.main 
  4. 访问 http://127.0.0.1:8080/develop/page/main#
